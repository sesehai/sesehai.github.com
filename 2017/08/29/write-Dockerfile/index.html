配置nginx的web服务

简介：

先使用python pip 包管理工具安装 supervisor, 使用supervisor管理nginx进程。

准备材料:

  • python pip，python包管理工具，安装需要： get-pip.py 文件。
    下载地址：get-pip.py
    详细安装请查看：install pip
  • supervisor,是在 UNIX-like 系统下管理进程的工具。
    需要配置文件：supervisord.conf
    详细文档请查看：supervisord
  • nginx，提供web服务。
    下载地址：nginx download
